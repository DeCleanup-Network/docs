# Developer Guide

Welcome to the DeCleanup Network developer documentation. This guide will help you understand our technical architecture and how to integrate with our platform.

## Tech Stack

- **Frontend**: React, Next.js
- **Smart Contracts**: Solidity
- **Backend**: Node.js, Express
- **Database**: PostgreSQL
- **Blockchain**: Ethereum & Layer 2

- JWT token management

### Verification System

- Image processing
- Proof validation

- $DCU Token
- Governance system
